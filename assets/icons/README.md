# App-ikoner

Denne mappen skal inneholde PNG-ikonene som brukes av manifestet:

- `icon-192.png` – 192×192 px
- `icon-512.png` – 512×512 px (kan også brukes som maskable)

Legg ferdiggenererte, kvadratiske PNG-er her før du tester PWA-installasjon. Filnavnene må matche de som er referert i `manifest.json`.
